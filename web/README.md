## Nomis

### Environment variables

You should add `NEXT_PUBLIC_API_HOST=http://localhost:3000` or `NEXT_PUBLIC_API_HOST=https://nomis.cc` to environment variables.

For local debugging:

- Create `.env.local` file
- Add necessary variable to the file

### Start

#### Dev

`
npm run dev
`

#### Build

`
npm run build
`

`
npm run start
`
